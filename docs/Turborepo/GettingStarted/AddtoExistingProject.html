兼容构建工具

将Turborepo添加到你现有的项目中 //Add Turborepo to your existing project

Turborepo可以在任何项目中使用，以加快package.json中脚本的执行速度。

在你安装了turbo之后，你就可以从turbo而不是软件包管理器中运行你所有的package.json任务。

通过正确配置你的turbo.json，你会注意到缓存何帮助你的任务运行得更快。

快速入门 //Quickstart

  1. 如果你还没有，请创建一个新的应用程序:

Next.js

npx create-next-app@latest

Vite

npm create vite@latest
  1. 安装 turbo:

npm

  npx create-next-app@latest

yarn

  yarn add turbo --dev

pnpm

  pnpm install turbo --save-dev
  1. 在你的新版本库的基础上添加一个 turbo.json 文件:

Next.js

filename="turbo.json"
  {
    "pipeline": {
      "build": {
        "outputs": [".next/**"]
      },
      "lint": {
        "outputs": []
      }
    }
  }

Vite

filename="turbo.json"
  {
    "pipeline": {
      "build": {
        "outputs": ["dist/**"]
      },
      "lint": {
        "outputs": []
      }
    }
  }

一些Vite启动器的package.json看起来像这样:

filename="package.json"
{
  "scripts": {
    "build": "tsc && vite build"
  }
}

我们建议将这些内容拆分为lint和build脚本。

filename="package.json"
{
  "scripts": {
    "build": "vite build",
    "lint": "tsc"
  }
}

这意味着Turbo可以单独安排它们。

  1. Try running build and lint with turbo:

npm

  npx turbo build lint

yarn

  yarn turbo build lint

pnpm

  pnpm turbo build lint

这将同时运行build和lint。

  1. 在不对代码做任何修改的情况下，尝试再次运行build和lint:

npm

  npx turbo build lint

yarn

  yarn turbo build lint

pnpm

  pnpm turbo build lint

你应该看到像这样的终端输出:

 Tasks:    2 successful, 2 total
Cached:    2 cached, 2 total
  Time:    185ms >>> FULL TURBO

恭喜 - 你刚刚在200毫秒内完成了构建和lint.

要了解如何实现这一点，请查看我们的核心概念文档

  1. 尝试用turbo运行dev:

npm

  npx turbo dev

yarn

  yarn turbo dev

pnpm

  pnpm turbo dev

你会注意到，你的dev脚本启动了。你可以用turbo来运行package.json中的任何脚本。
